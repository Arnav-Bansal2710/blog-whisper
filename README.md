# Blog Whisper

Blog Whisper is a simple blogging platform built using Node.js, Express, and EJS.  
Users can write blogs, view them in a feed, and explore content in a clean UI.

## Features

- Create blog posts
- View posts in a feed
- Multiple pages (Home, Feed, Writers, About, Contact)
- Responsive design using Bootstrap
- Server-side rendering with EJS

## Tech Stack

- Node.js
- Express.js
- EJS
- Bootstrap
- HTML/CSS

## Installation

1. Clone the repository

git clone https://github.com/Arnav-Bansal2710/blog-whisper.git
cd blog-whisper

2. Install dependencies

npm install

3. Start the server

node index.js

4. Open in browser

http://localhost:3000

## Note

Posts are stored in memory and will reset when the server restarts.
